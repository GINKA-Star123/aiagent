# aiagent

Unified AI VTuber runtime and control panel project.

## Scope

- `apps/`: runnable applications
- `aiagent/`: orchestration and core agent logic
- `integrations/`: wrappers around external providers and runtimes
- `domain/`: domain models and business rules
- `data/`: runtime data, character data, caches, logs
- `config/`: centralized settings
- `scripts/`: utility scripts
- `tests/`: test suites
- `docs/`: architecture and planning docs

## Status

Scaffold only. No legacy code has been migrated yet.
